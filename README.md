# Curso de Abaqus

Esboco de pagina para publicar um curso de Abaqus no GitHub Pages, com aulas, links de videos, PDFs e materiais complementares.

## Como usar

1. Substitua os links `SEU_VIDEO_01`, `SEU_VIDEO_02` etc. no `index.html` pelos links reais do YouTube, Vimeo ou Drive.
2. Coloque os PDFs na pasta `pdfs/`.
3. Coloque arquivos de apoio na pasta `arquivos/`.
4. No GitHub, va em `Settings > Pages` e publique a branch principal usando a pasta raiz do repositorio.

## Estrutura sugerida

```text
.
├── index.html
├── styles.css
├── pdfs/
│   ├── ementa.pdf
│   ├── aula-01-introducao.pdf
│   └── projeto-final.pdf
└── arquivos/
    └── modelos-exemplo.zip
```

## Personalizacao rapida

- Edite o titulo e a descricao principal no bloco `hero`.
- Adicione ou remova cards dentro da secao `Aulas do curso`.
- Atualize a carga horaria, nivel e formato no painel de resumo.
- Troque as cores principais no inicio do arquivo `styles.css`.
